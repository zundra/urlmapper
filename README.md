# URL Mapper URL Shortening Exercise

## Dependencies
    1.) Scala 2.11
    2.) Java 8
    3.) SBT 13.9
    4.) MongoDB

## Running
    1.) Naviate to $PROJECT_ROOT
    2.) sbt run

## Building a JAR
    1.) Naviate to $PROJECT_ROOT
    2.) sbt assembly

## Building a .deb
    1.) Naviate to $PROJECT_ROOT
    2.) debian-packaging/build.sh

## Usage
    1.) Navigate to http://localhost:9000
